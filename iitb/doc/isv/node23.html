Conclusions

  A new integrated stereo vision algorithm for estimating a dense disparity map has been formulated and implemented, using the intensity as the only cue. The stereo integration scheme is investigated in a monoresolution and multiresolution framework.

We would like to remark that the way the algorithm has been formulated, it lends itself to be implemented in parallel, using a Hopfield like neural network. Note that in all the energy modules, except the matching module, it is required that two binary fields (corresponding to the horizontal and vertical line fields) be updated, and this can be accomplished by using a Hopfield neural network. The matching block can also be put into the Hopfield neural network framework because the disparity is usually confined to a small range. For this purpose one could use a multilevel threshold function analogous to the multilevel sigmoidal function of [20].
