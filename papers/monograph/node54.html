Conclusions

The technique for maximizing the a posteriori distribution of an MRF to find the optimum set of interpretations uses a heuristic design rule for obtaining the clique functions. One of the disadvantages of this approach is that these clique functions can not be identified with intuitive dependencies. This renders the heuristics suspect. It is much more desirable if we could specify the domain knowledge in terms of dependencies instead of clique functions. The technique presented in this chapter accomplishes exactly this. The domain knowledge can be reliably constructed from the intuitive notions only.
